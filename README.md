# ReTech

A mobile application developed using Flutter and Firebase. An individual project done in year 2 semester 1 of polytechnic. This was done for the module 'Mobile App Development'.

The theme of this project is sustainability. I developed a mobile application where users can upload their faulty devices and seek help from other users.

Key features: Create/Read/Update/Delete 'Repair Request', Sign Up, Log in, Log in with Google, Forgot Password, Change Password, Change Font, Change Color Theme, Share, Web View, Direct to SMS App on Emulator, Set Notification.
